With this animation I wanted to make a hover mechanic for the squares by putting shadows below it so they look like they're floating. So I made them into figures of the size of 150 x 150. I used various hover mechanics like "top" to actually make the square float, and I used box shadows so the floating looks more natural. I used different number percentages for each square, so the lower squares float higher.
