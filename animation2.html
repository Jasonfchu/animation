With this animation I wanted to make a color pattern with squares. I first used keyframes to show where my animation would start and end. So I made them into figures of the size of 100 x 100. I made the animation set to “infinite” so the animation would go on forever, and I also added the tag “alternate” so the animations of the squares go back and forth smoothly.
